# 医伴医生端小程序

## 项目介绍

这是一个从Yiban小程序中分离出来的医生端小程序，包含医生登录、预约管理、处方管理等功能。

## 功能模块

- **登录模块**：医生账号登录
- **首页**：显示医生信息和统计数据
- **预约管理**：查看和处理患者预约
- **处方管理**：开具和管理处方
- **设置**：医生个人信息管理

## 技术架构

- **前端**：微信小程序原生开发
- **后端**：微信云开发
- **数据库**：云数据库

## 目录结构

```
│   ├── getAppointmentDetail/ # 获取预约详情
│   ├── getDoctorStats/      # 获取医生统计数据
│   ├── getPrescriptions/    # 获取处方列表
│   ├── getPrescriptionDetail/ # 获取处方详情
│   ├── updateAppointmentStatus/ # 更新预约状态
│   └── createTestDoctor/    # 创建测试医生账户
├── miniprogram/             # 小程序前端
│   ├── pages/               # 页面
│   │   ├── login/           # 登录页面
│   │   ├── homepage/        # 首页
│   │   ├── appointments/    # 预约管理
│   │   ├── prescriptions/   # 处方管理
│   │   └── settings/        # 设置页面

│   └── app.wxss             # 全局样式
└── project.config.json      # 项目配置
```

## 部署说明

1. 在微信开发者工具中导入项目
2. 部署云函数到云开发环境
3. 运行 `createTestDoctor` 云函数创建测试医生账户
4. 编译并预览小程序

## 注意事项

- 本项目使用微信云开发，需要配置云开发环境
- 首次使用需要先部署云函数
- 测试账号已内置，可直接登录使用

## 功能说明

1. **登录**：使用测试账号登录系统
2. **首页**：查看今日预约数和处方总数
3. **预约管理**：查看、确认或取消预约
4. **处方管理**：开具新处方，查看历史处方
5. **设置**：查看个人信息，退出登录
